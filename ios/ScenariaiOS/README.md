# ScenariaiOS - AI Image Generation App

Современное iOS приложение для создания персональных AI моделей и генерации изображений на основе Scenaria платформы.

## 🚀 Возможности

- 🧠 **Создание AI моделей** - Загрузка фотографий и создание персональной модели
- 🖼️ **Генерация изображений** - Создание изображений по текстовым описаниям
- 🎬 **Видео из изображений** - Конвертация изображений в видео (планируется)
- 💰 **Система оплаты** - Интеграция со Stripe для покупки кредитов
- 📱 **Современный UI** - SwiftUI с темной темой и градиентами
- 🔒 **Безопасность** - Keychain для хранения токенов, биометрия

## 📋 Требования

- iOS 16.0+
- Xcode 15.0+
- Swift 5.9+

## 🏗️ Архитектура

Проект построен на принципах **Clean Architecture** и **MVVM**:

```
ScenariaiOS/
├── Source/                    # Исходный код
│   ├── App/                   # Конфигурация приложения
│   ├── Core/                  # Ядро системы
│   │   ├── Network/           # Сетевой слой
│   │   ├── Storage/           # Хранение данных
│   │   ├── Security/          # Безопасность
│   │   └── Utils/             # Утилиты
│   ├── Features/              # Функциональные модули
│   │   ├── Authentication/    # Авторизация
│   │   ├── ModelCreation/     # Создание моделей
│   │   ├── ImageGeneration/   # Генерация изображений
│   │   ├── Gallery/           # Галерея
│   │   ├── Profile/           # Профиль
│   │   └── Payment/           # Платежи
│   └── Shared/                # Общие компоненты
│       ├── UIComponents/      # UI компоненты
│       ├── Styles/            # Стили и темы
│       ├── Constants/         # Константы
│       └── Models/            # Общие модели
├── Resources/                 # Ресурсы
│   ├── Assets/                # Изображения, иконки
│   ├── Fonts/                 # Шрифты
│   ├── Sounds/                # Звуки
│   └── Localization/          # Локализация
├── Configuration/             # Конфигурация
│   ├── Environments/          # Настройки сред
│   └── Info/                  # Info.plist файлы
├── Tests/                     # Тесты
│   ├── Unit/                  # Unit тесты
│   ├── UI/                    # UI тесты
│   └── Integration/           # Интеграционные тесты
├── Documentation/             # Документация
├── Tools/                     # Инструменты разработки
└── Scripts/                   # Скрипты автоматизации
```

## 🛠️ Настройка проекта

1. **Клонирование репозитория:**
```bash
git clone <repository-url>
cd ScenariaiOS
```

2. **Открытие в Xcode:**
```bash
open ScenariaiOS.xcodeproj
```

3. **Настройка конфигурации:**
   - Отредактируйте файлы в `Configuration/Environments/`
   - Укажите URL вашего API сервера
   - Настройте Bundle ID и команду разработчика

4. **Установка зависимостей:**
   - Проект использует Swift Package Manager
   - Зависимости автоматически загрузятся при первой сборке

## 🔧 Технологии

- **SwiftUI** - Декларативный UI фреймворк
- **Combine** - Реактивное программирование
- **Swift Package Manager** - Управление зависимостями
- **URLSession** - Сетевые запросы
- **Keychain Services** - Безопасное хранение
- **Core Data** - Локальная база данных (опционально)
