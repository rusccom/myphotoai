# MyPhotoAI iOS App

Современное iOS приложение для генерации изображений с помощью AI, написанное на Swift и SwiftUI.

## Возможности

- 🎨 Генерация изображений по персональным AI моделям
- 📝 Генерация изображений из текстовых описаний  
- 👔 Виртуальная примерка одежды (скоро)
- 🔍 Улучшение качества изображений (скоро)
- 🎬 Создание видео из изображений (скоро)
- 🌙 Современный темный дизайн с неоновыми акцентами
- 📱 Оптимизировано для iPhone

## Требования

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+

## Настройка

1. Откройте проект в Xcode:
   ```bash
   cd ios/MyPhotoAi
   open MyPhotoAi.xcodeproj
   ```

2. Измените URL бекенда в файлах:
   - `Managers/AuthManager.swift` - строка 10
   - `ViewModels/ModelGenerationViewModel.swift` - строка 9
   - `ViewModels/GalleryViewModel.swift` - строка 13
   
   Замените `http://localhost:5001` на URL вашего сервера.

3. Настройте Bundle ID и подписание:
   - Выберите проект в навигаторе
   - Перейдите в Signing & Capabilities
   - Выберите свою команду разработчика
   - Измените Bundle Identifier на уникальный

## Запуск

1. Выберите симулятор или подключенное устройство
2. Нажмите Cmd+R или кнопку Run в Xcode
3. При первом запуске появится onboarding с инструкцией

## Архитектура

- **SwiftUI** - декларативный UI фреймворк
- **MVVM** - паттерн Model-View-ViewModel
- **Async/Await** - современный подход к асинхронности
- **Keychain** - безопасное хранение токенов

## Дизайн

Приложение использует темную тему с неоновыми акцентами:
- Основной цвет: Purple (#8B5CF6)
- Дополнительные: Blue, Orange, Green, Indigo
- Фон: Градиент от черного к темно-серому

## API Endpoints

Приложение работает с следующими API:
- `/api/auth/login` - вход
- `/api/auth/register` - регистрация
- `/api/models` - список AI моделей
- `/api/generation/lora-generate` - генерация по модели
- `/api/generation/start` - текст в изображение
- `/api/generation/history` - история генераций

## Безопасность

- Токены хранятся в Keychain
- Поддержка биометрии (Face ID/Touch ID) - планируется
- SSL pinning - планируется

## Известные проблемы

- Для работы с localhost необходимо разрешение в Info.plist
- На реальном устройстве требуется HTTPS соединение
