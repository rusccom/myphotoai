# Инструкции по добавлению изображений пресетов

## 📁 Структура папок

Ваша структура уже создана:

```
public/media/presets/
├── presets-config.json     (конфигурационный файл)
├── README.md               (общая информация)
├── INSTRUCTIONS.md         (этот файл)
├── Portraits/              (папка для портретов)
├── Fashion/                (папка для модных фото)
├── Professional/           (папка для профессиональных фото)
└── Creative/               (папка для креативных работ)
```

## 🖼️ Требования к изображениям

### Формат и размер
- **Соотношение сторон:** 3:4 (вертикальный)
- **Рекомендуемое разрешение:** 600x800 px или выше
- **Форматы:** JPG, PNG, или WebP
- **Размер файла:** желательно до 500KB для быстрой загрузки

### Названия файлов
Каждая категория должна содержать 5 файлов с именами:
- `1.jpg`
- `2.jpg`
- `3.jpg`
- `4.jpg`
- `5.jpg`

## 📝 Как добавить изображения

### Шаг 1: Подготовьте изображения
1. Выберите 5 изображений для каждой категории (всего 20 изображений)
2. Убедитесь, что они в формате 3:4 (вертикальные)
3. Оптимизируйте их размер (рекомендуется 600x800px)

### Шаг 2: Переименуйте файлы
Переименуйте файлы согласно схеме: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`

### Шаг 3: Разместите в папки
Разместите по 5 изображений в каждую папку категории:

**Portraits/** - портретные фото (5 файлов)
- Пример контента: классические портреты, headshots, студийные фото людей

**Fashion/** - модные фото (5 файлов)
- Пример контента: модельные фото, fashion съемка, стильные образы

**Professional/** - профессиональные фото (5 файлов)
- Пример контента: деловые портреты, корпоративные фото, LinkedIn стиль

**Creative/** - креативные работы (5 файлов)
- Пример контента: художественные фото, необычные ракурсы, арт-стиль

## 🔄 Автоматический fallback

Если изображения не будут добавлены:
- Компонент автоматически покажет цветные плейсхолдеры
- Каждая категория будет иметь свой цвет плейсхолдера
- Функциональность фильтрации будет работать

## 🎨 Рекомендации по подбору

1. **Единый стиль** - старайтесь подбирать изображения в едином стиле для каждой категории
2. **Качество** - используйте высококачественные изображения
3. **Репрезентативность** - изображения должны отражать возможности вашего AI
4. **Разнообразие** - в рамках категории показывайте разные варианты

## 🔧 Добавление новой категории

Если хотите добавить новую категорию:

1. Создайте новую папку в `public/media/presets/`
2. Добавьте 5 изображений (1.jpg - 5.jpg)
3. Обновите `presets-config.json`:

```json
{
  "categories": ["All", "Portraits", "Fashion", "Professional", "Creative", "YourNewCategory"],
  "presets": [
    ...существующие категории...,
    {
      "category": "YourNewCategory",
      "title": "Your Category Title",
      "images": ["1.jpg", "2.jpg", "3.jpg", "4.jpg", "5.jpg"]
    }
  ]
}
```

## ⚙️ Изменение количества изображений

Если хотите показывать больше или меньше 5 изображений:

1. Добавьте нужное количество файлов в папку
2. Обновите массив `images` в `presets-config.json`
3. CSS автоматически адаптирует grid

Пример для 10 изображений:
```json
{
  "category": "Portraits",
  "title": "Portrait Presets",
  "images": ["1.jpg", "2.jpg", "3.jpg", "4.jpg", "5.jpg", "6.jpg", "7.jpg", "8.jpg", "9.jpg", "10.jpg"]
}
```
