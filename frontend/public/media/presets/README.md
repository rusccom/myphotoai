# Presets Structure

This folder contains preset images for the homepage Capabilities section.

## Structure

```
presets/
├── Portraits/      (5 images in 3:4 format)
├── Fashion/        (5 images in 3:4 format)
├── Professional/   (5 images in 3:4 format)
└── Creative/       (5 images in 3:4 format)
```

## Image Requirements

- **Aspect Ratio:** 3:4 (vertical format)
- **Recommended Size:** 600x800px
- **Formats:** JPG, PNG, WebP
- **File Names:** `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`

## How to Add Images

1. Place 5 images in each category folder
2. Name them: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`
3. Ensure they are in 3:4 vertical format
4. If no images are present, placeholders will be displayed automatically

## Configuration

Edit `presets-config.json` to manage categories and image filenames.
