# Model Generation Block Media

Эта папка содержит изображения для блока "Генерация фото по обученной модели" на главной странице.

## Структура папок

```
model-generation/
├── main/           # Главное большое фото (слева, выровнено по центру)
│   └── main.jpg
└── grid/           # 12 фото для сетки 4x3 (справа, 4 колонки × 3 ряда)
    ├── image-1.jpg
    ├── image-2.jpg
    ├── image-3.jpg

```

## Как добавить свои фото

### 1. Главное фото
Поместите ваше изображение в папку `main/` с именем `main.jpg`

**Рекомендации:**
- Формат: JPG, PNG, WEBP
- Соотношение сторон: 3:4 (например, 600x800px или 900x1200px)
- Размер файла: до 500KB
- Качество: высокое разрешение
- Содержание: лучший результат генерации AI модели

### 2. Сетка фото
Поместите 12 изображений в папку `grid/` с именами от `image-1.jpg` до `image-12.jpg`

**Рекомендации:**
- Формат: JPG, PNG, WEBP
- Соотношение сторон: 3:4 (например, 300x400px или 450x600px)
- Размер файла: до 200KB каждое
- Качество: среднее-высокое разрешение
- Содержание: различные стили/позы/сценарии генераций

## Заглушки

Если фото отсутствуют, блок автоматически покажет заглушки (placeholder).
Это позволяет видеть структуру блока до загрузки реальных изображений.

## Анимации

- **Главное фото**: появляется первым при скролле (fade-left анимация)
- **Сетка фото**: появляются хаотично с разными задержками (scale анимация)
- **Hover эффекты**: увеличение, тени, glassmorphism

- Имена файлов должны точно соответствовать указанным выше
